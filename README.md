# Namka Portfolio - Ali Mora

A Next.js 14 personal portfolio website for Ali Mora, featuring multiple live AI demos powered by the Gemini API.

## Demos

The portfolio contains several AI prototypes in the `app/demos` directory:
- AI Chef
- AI Mentor Matcher
- AI Website Roaster
- Developer Brief Generator
- NFT Describer (Vision)
- Weather Advisor

## Running Locally

1. Create a `.env.local` file with the required variables (e.g. `GOOGLE_AI_API_KEY`, `RESEND_API_KEY`).
2. Run `npm install`
3. Run `npm run dev`

The app runs on [http://localhost:3000](http://localhost:3000).

## Build

To build for production, run:
```bash
npm run build
```
